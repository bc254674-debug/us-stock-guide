股票入门基础 — 看懂行情数据

K线图 · 开盘价 · 前收盘 · 成交量 · 日均波动 · MA5 / MA10

什么是K线图？

K线图（Candlestick Chart）是股票市场最常用的价格走势图表，起源于18世纪日本的大米交易市场。一根K线包含了开盘价、收盘价、最高价、最低价四个关键价格信息。

阳线（涨）
— 最高价 — — 收盘价 ↑ — — 开盘价 — — 最低价 —
阴线（跌）
— 最高价 — — 开盘价 — — 收盘价 ↓ — — 最低价 —
一段包含阳线、阴线、十字星的迷你K线走势 — 整体呈上升趋势
长阳线
实体很长 → 买方力量极强，涨势明确
长阴线
实体很长 → 卖方力量极强，恐慌抛售
十字星
实体极短+长影线 → 多空纠结，方向不明
  • 阳线（红色实体）：收盘价 > 开盘价，当天上涨。A股红色涨，美股绿色涨。
  • 阴线（绿色实体）：收盘价 < 开盘价，当天下跌。
  • 实体长度：越长说明推动力越强。长阳线=资金抢筹；长阴线=恐慌出逃。
  • 影线长度：上影线长=上方抛压重(冲高回落)；下影线长=下方有支撑(探底回升)。
  • 十字星：开盘≈收盘，多空双方势均力敌，变盘前兆。
新手重点看：实体颜色（涨跌）→ 实体长度（力度）→ 影线长度（阻力/支撑）。三者结合判断。

关键价格概念

一根阳线的完整解剖：开盘 → 冲高 → 回落 → 收盘。每根K线都在讲述当天多空博弈的故事。
📌

开盘价（今开）

每个交易日开始时的第一笔成交价格。A股9:30开盘，开盘价由集合竞价（9:15-9:25）产生，反映市场隔夜消化信息后的初始定价。

开盘 > 前收盘（高开）→ 利好；开盘 < 前收盘（低开）→ 利空。
📌

前收盘价（昨收）

上一交易日的最后一笔成交价格，是判断今日涨跌的基准参照点。所有涨跌额和涨跌幅都以它为基准。

涨跌额 = 当前价 − 前收盘
涨跌幅 = 涨跌额 ÷ 前收盘 × 100%
📌

最高价 / 最低价

最高价是当天出现过的最高成交价；最低价是当天出现过的最低成交价。两者差值反映日内波动范围。

最高−最低差距大 → 多空博弈激烈。差距小的股票更适合保守型投资者。
📌

当前价（最新价）

当前市场上最新一笔成交的价格。交易时段内实时变动，收盘后即为当日收盘价。下单成交价通常接近当前价。

美股盘后交易（After Hours）也会产生价格变动，但流动性通常较低。

成交量 — 市场的"温度计"

成交量是某一时间段内成交的股票数量。它是判断趋势可靠性的核心指标——没有成交量支撑的价格变动往往不可持续。

价格上行（蓝线）+ 成交量放大（红柱）→ 上涨有量能支撑，趋势健康可靠

放量上涨 ✅

📈📊

价格上涨 + 成交量放大 → 大量资金买入，趋势可靠，多方力量强劲。

放量下跌 ⚠️

📉📊

价格下跌 + 成交量放大 → 资金恐慌出逃，空方占主导，远离为妙。

缩量上涨 🤔

📈🔇

价格上涨 + 成交量萎缩 → 买盘不足，"强弩之末"，警惕回调。

缩量下跌 🔍

📉🔇

价格下跌 + 成交量减少 → 卖盘衰竭，下跌动能减弱，关注机会。

核心原则："量在价先"——成交量的变化往往领先于价格变化。放量突破关键价位通常是可靠的交易信号。

日均波动幅度 — 衡量风险与机会

日均波动幅度是当日最高价与最低价的差值占前收盘价的比例，直接反映股票的价格活跃度和潜在风险。

日均波动幅度 = (最高价 − 最低价) ÷ 前收盘价 × 100%
同一资金，高波动股一天可能赚/亏4%，低波动股只波动1%左右。新手建议从低波动入手。

高波动股票（>3%）

科技成长股、小盘股、题材股。日内波动可达3%-5%以上。潜在收益高但风险大，一天可能亏掉一周涨幅。适合能承受较大回撤的投资者。

低波动股票（<2%）

大型银行股、公用事业、消费蓝筹。日内波动约1%-2%，价格稳定。适合保守型投资者和长期持有者，也是新手入门最佳选择。

新手建议：从标普500 ETF（VOO/SPY）开始——波动适中、覆盖面广、历史悠久。

移动平均线 — MA5 与 MA10

移动平均线（Moving Average）将一段时间内的收盘价加总求平均，连成平滑曲线。它过滤掉每日噪音，让你看清真正的价格趋势。

MA5（蓝线）上穿MA10（紫线）= 金叉买入；MA5下穿MA10 = 死叉卖出。注意价格噪音被均线完美过滤。
MA5

5日均线 · 周线

取最近5个交易日收盘价的平均值。A股一周5天，所以MA5也叫"周线"。对价格变化反应最灵敏，适合判断短期方向。

MA5 = (D₁ + D₂ + D₃ + D₄ + D₅) ÷ 5
MA10

10日均线 · 双周线

取最近10个交易日收盘价的平均值，覆盖两周数据。比MA5更平滑稳定，反映中短期趋势，是短线交易最常用的参考线。

MA10 = (D₁ + D₂ + … + D₁₀) ÷ 10

金叉与死叉 — 最经典的买卖信号

金叉 Golden Cross

MA5 从下方向上穿越 MA10 → 短期趋势转强 → 买入/加仓信号

死叉 Death Cross

MA5 从上方向下穿越 MA10 → 短期趋势转弱 → 卖出/减仓信号

重要提醒：金叉死叉是辅助参考，不是100%准确的信号。震荡市中可能频繁出现假信号。务必结合成交量、市场环境、基本面综合判断。

综合应用 — 五步看懂一只股票

五步分析流程：从趋势到决策，层层递进
  1. 第一步：看K线图整体趋势

    先看中长期走势是上升、下降还是横盘。一浪比一浪高=上升趋势；一浪比一浪低=下降趋势。单日涨跌可能是噪音，趋势才是信号。

  2. 第二步：看MA5和MA10的位置关系

    MA5在MA10上方且两者向上 → 短期向好，持股或买入。MA5在MA10下方且向下 → 短期偏弱，观望或减仓。两线纠缠 → 震荡市，方向不明。

  3. 第三步：看今日开盘价与前收盘价

    高开 → 市场情绪积极，可能有催化剂。低开 → 可能有利空。平开 → 市场在等待更多信息释放。

  4. 第四步：看成交量配合情况

    涨+放量=强势；涨+缩量=弱势。跌+放量=恐慌，远离；跌+缩量=调整尾声，可关注。缩量突破关键价位=假突破概率大。

  5. 第五步：看日均波动幅度

    波动大 → 风险机会都大，适合短线。波动小 → 稳定，适合长线。新手从日均波动<2%的蓝筹股/指数ETF开始，逐步积累经验。

一句话总结：K线趋势定方向 → 均线关系定时机 → 成交量确认真假 → 波动幅度量风险。四者结合，理性决策。
